component overview

The following table shows a list of all used components for this microscope. For a better overview it is split in the main cube, the excitation path, the emission path, the laser group and the sample holder. Some of their parts were already present like the fiber coupler, so I do not have a price for them. Additionally you need some post material for mounting the optical components in the detection beampath and the laser assembly.

Notice: All prices are without taxes and serve only for an overview.
